PROJEKTBEZEICHNUNG: 3wins
PROJEKTZWECK: Info Projekt Gymnasium Dorfen 10 Klasse
DATUM: 23. Juni 2019
WIE IST DAS PROJEKT ZU STARTEN: 
    mittels BlueJ: Neue Game Instanz mit Parametern calculationDepth(8-16) und youStart(true/false)
    via Console z.B.: java Game 8 true
WIE KANN DAS PROJEKT GETESTET WERDEN:
    mittels BlueJ: Klasse GameTest, "Test All" wählen
AUTOR(EN):
    Andi Neudecker, Peter Schuster, Nils Münchhausen
BENUTZERHINWEISE:
    Für hohe calculationDepth Werte (>12) ist ein guter Rechner nötig, da die Berechnungszeit zu lange dauert.
SOURCE REPOSITORY:
